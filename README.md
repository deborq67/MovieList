# MovieList
A simple interface to add movie titles.

![MovieList.png](https://i.ibb.co/pvmKx93s/Movie-List.png)

This application is a simple and effective way of keeping track of the movies you want to watch, and removing the movies you have already watched.

* **Add and Remove Movies:** Hit the corresponding button and it will be done for you.

* **Persistent Memory:** Even if you reload or exit, the application will remember the movies you stored.

* **Sort your Movies:** Even if you reload or exit, the application will remember the movies you stored.

## How to Run

1. Download the repository as a ZIP and extract that folder.

2. Open `index.html` on a browser. That's it!
